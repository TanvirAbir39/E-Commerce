# E-Commerce-website-using-html-css-Js
  A simulation of client side and Admin side Upsate Product from e-commerce website with feature as add to cart, proceed for checkout options. It simultes a shopping cart within a website.

Follow the following steps to view the project:

1. Command (CMD) call (node app) press Enter key. 
   
2. Show the Server Localhost Address (http://localhost:6060) copy this localhost address.

3. Paste the address Any WebBrowser.

Now you are all set to view the  E-Commerce-website project.
